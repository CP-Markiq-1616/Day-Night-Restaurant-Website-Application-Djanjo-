**Welcome To The Day & Night Restaurant Website**

Please use the following link:

For best results please use a Windows computer with Google Chrome.




Key Points


**Number 1:**

When you are finished adding your items to the cart from the "Order Take Out" screen, press the "View Total" button to view your total before proceeding to checkout. 


<img width="600" height="600" alt="View Total Readme" src="https://github.com/user-attachments/assets/72e7e655-b9d5-4926-8913-7b9683e69026" />


**Number 2:**

You do not have to use a real email address to sign up.

Username can be a name and a number such as "Bob_User789" for example.

The email address can be something like "Bob_User789@yahoo.com". 

Please create a medium strength password and write it down. The backend system will not accept a password if it is too simple.

Once logged in with username and password, you will be able to view your order history if you have any previous orders.


**Number 3:**

The "Forgot Password" button is fully functional.

After clicking forgot password at the login screen, you will be prompted to enter the email address that was used to sign up.

Once the email address is entered, click "Reset My Password". You will now see a page that says "Password Reset Sent".

At this point, the link will come to the backend server for only the developer to see. 

**Please contact me directly by email (Markiq1616@gmail.com) for the reset link, or it can be done via live demonstration on a video call. Otherwise, simply create a new account with a new password.**

If you have retreived the link from me, paste it into your browser and you will be prompted to create a new password.

Once the new password is set, you will be prompted to log in.
